# Maximum Carnage GBC — Controls Reference

## All Characters

| Button | Ground | Air |
|--------|--------|-----|
| D-Pad | Move / Crouch | Steer |
| A | Jump | Spider-Man: **Attach web-swing** |
| B | Attack (punch/kick) | Attack |
| A+B | **Primary Special** | Primary Special |
| SELECT | **Cycle ability slot** | Cycle ability slot |
| START | Pause | Pause |

---

## Spider-Man: Web-Swing (ANYWHERE — Arcade Feel)

1. **Jump** — press A on ground
2. **Attach web** — press A while airborne → pendulum swings from invisible anchor above
3. **Steer** — hold D-Pad Left/Right to pump and steer the arc
4. **Launch** — release A to detach → flies forward with momentum
5. **Chain** — immediately press A again mid-air to reattach for multi-swing combos

> **Pro tip:** Release A at the peak of the arc (horizontal position) for maximum
> launch height and distance.

## Ability Slots (SELECT to Cycle)

The HUD bottom-left shows:  `[ICON]  ● ● ○ ○`
- **Filled dots** = unlocked slots
- **Lit/bright dot** = currently active
- **Icon** = active ability picture

Press SELECT to cycle. Then press A+B to fire the active ability.
